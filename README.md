# Hath Exchange OTEL Exporter

The Hath exchange in a certain H website is a market of offers and demands.
However, there is no way to track past offers and demands.
This exporter exports every data from _that_ page, to be used for tracking and "investment".

Yes, it is vibe-coded, and no I won't explain it.

The implementation is simple and I'm lazy.

## Usage

Just run it on linux. Access on port 9101, path /metrics.

You need to pass a valid Netscape cookie file.

## Build

```bash
make
```
